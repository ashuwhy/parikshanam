Full Platform Design

Two new apps — here's everything we're building. Review and confirm before I write the spec.

⬡ ADMIN
admin.parikshanam.com
apps/admin
Sidebar Sections
📊 Dashboard
Total students, teachers, courses, revenue. Pending course approvals. Recent purchases.
📚 Courses
Create/edit courses. Approve teacher-submitted courses. Manage modules, lessons, videos, quizzes. Assign teachers.
👨‍🎓 Students
Browse all students. Search by name/phone. Edit email. View purchases & progress. Deactivate account.
👨‍🏫 Teachers
List teachers. Invite new via email. View their courses. Edit email. Deactivate.
💳 Purchases
All transactions. Filter by date/course/status. Razorpay payment IDs. Revenue totals.
⚙️ Settings
Manage class levels, olympiad types. Platform config.
Auth
Email + password via Supabase. Role check: profiles.role = 'admin'. Service-role client for all DB writes (bypasses RLS).
📖 TEACHER
teach.parikshanam.com
apps/teacher
Sidebar Sections
📊 Dashboard
My course stats, enrolled students, pending approvals, recent activity.
📚 My Courses
Create new courses (submitted for admin approval). Manage modules, lessons, video uploads, quizzes. See approval status.
👨‍🎓 My Students
Students enrolled in their courses. Progress per lesson/quiz. Read-only.
👤 Profile
Update name, email, avatar, bio.
Auth — Invite-Only Flow
1. Admin invites teacher by email →
2. Email sent with secure token link →
3. Teacher sets password at /invite?token=xxx →
4. Supabase user created with role='teacher' →
5. Lands in teacher dashboard
🗄️ Database Changes (new migration)
profiles (update)
+ role: 'student' | 'teacher' | 'admin'
+ bio text
+ is_active boolean
teacher_invites (new)
id, email, token (unique)
invited_by (admin uuid)
expires_at, accepted_at
course_teachers (new)
course_id + teacher_id (PK)
Links teachers to courses
assigned_by (admin uuid)
courses (update)
+ status: 'draft' | 'pending_review' | 'active' | 'archived'
+ created_by (uuid)
RLS Policies (update)
Admin: service-role bypasses all
Teacher: manage own courses
Teacher: view enrolled students
Tech Stack
Next.js 16 + Tailwind v4
Supabase JS + service role
Resend (email invites)
React Hook Form + Zod
